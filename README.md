# JS-Clock-Chrome-App
- 인원: 이혜성
- 기간: 2021.05.18. - 2021.05.19.
- 내용: Vanila Javascript를 이용한 Chrome Webbrowser의 앱플리케이션 구현
  - 현재 시간 표기
  - ToDo 리스트 기능
  - 외부 API를 이용하여 fech한 GeoData 화면에 나타냄(온도, 위치)
  - 랜덤한 배경 이미지를 랜더링 할 때마다 나타냄
